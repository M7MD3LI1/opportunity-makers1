# صناع الفرص — Opportunity Makers Platform

منصة متكاملة لإدارة المتطوعين واللجان، مبنية باستخدام تقنيات Full-Stack حديثة.

## التقنيات المستخدمة (Tech Stack)

### الواجهة الأمامية (Frontend)
- **React 19** مع **TypeScript**.
- **Vite** للتطوير والبناء السريع.
- **Tailwind CSS** للتنسيق الجمالي.
- **Framer Motion** للرسوم المتحركة.

### الواجهة الخلفية (Backend)
- **Node.js** و **Express**.
- **Prisma ORM** للتعامل مع قاعدة البيانات.
- **TypeScript** لضمان جودة الكود.

## طريقة التشغيل (Setup)

1. قم بتثبيت التبعيات في كلا المجلدين (backend و frontend):
   ```bash
   npm install
   ```

2. قم بتشغيل المشروع باستخدام ملف الاختصار الذي أنشأناه:
   - على Windows: قم بتشغيل `start.bat`
   - على Linux/Mac: قم بتشغيل `start.sh`
